# HTB Lab

[Hack The Box](https://app.hackthebox.com/home) 是一个在线渗透测试平台，提供大量退役和活跃的虚拟机靶机，涵盖 Web 应用、网络渗透、权限提升等多个方向。**注意：仅记录退役靶机的 Writeup，遵守平台规则。**

## 平台特色

- **靶机丰富**：提供大量退役和活跃的虚拟机靶机，持续更新
- **实战性强**：真实模拟渗透测试场景，提升实战能力
- **方向全面**：涵盖 Web 应用、网络渗透、权限提升、Active Directory 等多个方向
- **社区活跃**：拥有活跃的社区，可以交流学习经验
- **难度分级**：靶机按难度分级，适合不同水平的渗透测试人员
